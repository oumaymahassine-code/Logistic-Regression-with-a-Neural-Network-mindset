# Logistic Regression as a Neural Network Mindset

Ce projet illustre la régression logistique en utilisant une approche inspirée des réseaux de neurones.

## Contenu

- `Logistic.ipynb` : Notebook principal du projet.
- `lr_utils.py` : Fonctions utilitaires pour la régression logistique.
- `datasets/` : Données d'entraînement et de test (`train_catvnoncat.h5`, `test_catvnoncat.h5`).
- `images/` : Images utilisées dans le notebook pour les exemples et visualisations.

## Objectif

- Comprendre la régression logistique.
- Implémenter les étapes d'un modèle binaire de classification.
- Visualiser la propagation avant, la rétropropagation et la mise à jour des paramètres.

## Exécution

1. Ouvrir `Logistic.ipynb` dans Jupyter Notebook ou JupyterLab.
2. Exécuter les cellules séquentiellement.
3. Vérifier les résultats et les graphiques générés.

## Prérequis

- Python 3
- Jupyter Notebook
- NumPy
- Matplotlib
- H5py
